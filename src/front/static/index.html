Air Traffic Control Simulator

  • 3D Air Traffic Control Simulator created with C++ and OpenGl.
  • Designed so multiple aircrafts appear randomly at different times and places near the airport.
  • Created an algorithm that designed an efficient path for each aircraft to land as fast as possible, evading collision and storms.
  • To better simulate real scenarios, each path was heavily restricted to follow the navigation rules that real aircrafts are subjected to: minimum distance between aircrafts, maximum and minimum speed at landing, maximum acceleration and deceleration, etc.
  • The algorithm created and implemented can coordinate the trajectories of more than 40 aircrafts without any collisions.

  • C++ and OpenGL
  • OOP
  • Trajectory Planning
